1.短愚蛮怒について

  短愚蛮怒は変愚蛮怒のバリアントです。

  コンセプトは ”サクサク” ”スピーディー” ”楽しい” です。

  変愚蛮怒の楽しさはそのままに、短時間で***勝利***できるように変更されています。

  主な変更点は
  鉄獄を25階に短縮（名前は短獄に。他のダンジョンも短縮。モンスターのレベルと出現階数も変更）
  鉄獄の全体マップ上の位置変更（最初の村のすぐ西に）
  鉄獄のフロアマップが変愚に比べて小さい（探索効率を上げるためイークの洞窟と同じサイズに変更）
  ランダムクエストは2階から20階までの偶数階に
  モンスターを倒した時の経験値が変愚に比べて10倍
  高級品のアイテムの出現率が高い。（ユニークのドロップはほぼ高級品確定）
  中盤を楽に進めるために変更された新しく強力なエゴアイテム

  また、作者が感じる変愚の不満点を解消しています。

  初めにランタンが売っていない
  ワーグ探すのが面倒
  初期の探索に必要なアイテム買う金が足りない
  街の移動が以下略
  鉄獄行くのが…
  耐性が揃わなくて深層に行けない
  スピードが足りなくて…
  オベロン倒したあとの物資集めが面倒

  かなりご都合主義なところはありますが、
  サクサク、スピーディー、楽しい、のコンセプトを優先しています。

  変愚勝利者でしたら5時間程度、上級者でしたら2時間以内でも勝利は可能かと思います。

3.プロローグ

  「グアァァァ！」

  混沌のサーペントの断末魔が変愚蛮怒の世界に響き渡った。
  サーペントは消滅し変愚蛮怒の世界は平和を取り戻したかに見えたが…

  しかし、サーペントは消滅する前に最後の力を振り絞り
  シャドウシフトにより別世界に転生を行っていたのだ。

  ***勝利者***となった＠は違和感を感じアンバライトのトランプ魔術師に調査を依頼した。
  綿密な調査の結果、サーペントは短愚蛮怒の世界に転生し力を取り戻しつつあることがわかった。

  更に、先に異変を感じていたオベロンは短愚蛮怒の世界に影を派遣し
  力を取り戻しつつあるサーペントを守護しているようだ。

  ***勝利者***となった＠だが、短愚蛮怒の世界に転生したサーペントを倒すために
  変愚世界での名声、栄誉、経験をすべて捨て去り、単身、短愚蛮怒の世界に転生することを決意した。

  「……ここは？」
  見慣れた辺境の村の入口のようだが、
  おや、ちょっと待てよ？…



4.基本的なプレイ

  標準の設定ならば、テンキーで移動しフルキーでコマンド入力を行います。
  コマンドが分からない場合、リターンキー又は'x'キーを押すとコマンドメニューが
  出るので、そこから選んで実行することができます。
  コマンドの最後についているアルファベットはコマンドを実行するためのキーです。
  例えば、「休息する(R)」は'R'を押すことでも実行でき(大文字・小文字は区別)、
  「周りを調べる(l/x)」は標準のキー配置なら'l'、ローグライクキー配置なら'x'を
  押すことでも実行できます。

  最初は変愚と同じように序盤の2つのクエストからスタートすることをオススメします。
  クエストをこなすことで序盤に必要な物資が揃うでしょう。
  
  次はワーグのクエストを受けてから、短獄の村のすぐ隣りにある「イークの洞穴」に行くことをお勧めします。
  変愚と違い1画面のみと大変狭くなっています。また、地下３階までしかありません。
  地下2階のクエストでワーグが見つからなくて困ることはないでしょう。
  クリア出来て余裕があるなら地下３階まで足を伸ばしても良いかもしれません。
  ここは最深階に住んでいる「イークの大王」と取り巻きの召喚が強敵です。
  無理して倒す必要はないので状況に応じて撤退することも必要です。
  
  短獄の村に戻ったらクエストの報酬と物資を補充して短獄に向かいましょう。
  短獄は短獄の村のすぐとなりの西側に位置します。すぐに見つかるでしょう。
  
  気が向いたら街の外へ出ていろいろなダンジョンへ潜ってみるといいでしょう。
  街の外へ出るときは、多くの食料を持っていかなければいけません。
  この世界は非常に広いので、街の外に出るときは'<'で広域マップに入り、
  目的地についたところで'>'で通常マップに戻る、という移動の仕方が便利です。
  荒野では道や草原は比較的安全ですが、溶岩地帯や海の上は強力なモンスターが
  いて危険です。こういったところへ行くためにはさらに強くならないといけません。

  短獄の村と同じ島にある火山は駆け出しの冒険者ではまだまだ歯が立ちません。
  もっと強くなってから戻ってくるといいでしょう。
  東に行くと短獄の村とモリバントの中間付近にオークの住む洞窟があり、
  その南の島にはミノタウロスが住むと噂される魔法の迷宮があります。
  他にも多くのダンジョンが勇敢な冒険者が訪れるのを待っています。
  
  このゲームのメインとなるダンジョンであり、最終目的である
  「混沌のサーペント」の棲む「短獄」のダンジョンは
  短獄の村のすぐとなり、西側に位置します。

  その他のくわしい操作方法などは readme_hengband.txt を参照してください。


5.インストール

●Windows
  
  ローグライク実験室さんでバイナリを用意していただきました。感謝。
  
  ローグライク実験室
  https://dl.dropboxusercontent.com/u/26625628/index.html#tanguband
  
  
●UNIX,Macintosh

  変愚蛮怒のコンパイルを参考に書いています。
  （正確かは自信ないです。）

  X11環境が整っているならば
  ZIPをダウンロードして展開後ディレクトリに移動して
  ./bootstrap
  ./configure
  make install
  でインストールできます。

  変愚蛮怒スポイラーの"コンパイル/Mac"のページを参考にしてください。
  http://mars.kmc.gr.jp/~dis/heng_wiki へどうぞ。

  後は、ASCII文字画面なら、
  ./tanguband -- -n＜ウィンドウの数＞  
  タイルグラフィックス画面なら、
  ./tanguband -g -- -n＜ウィンドウの数＞      ## 通常の8x8タイル
  ./tanguband -g -- -b -n＜ウィンドウの数＞   ## 2倍幅サイズのタイル
  ./tanguband -g -- -a -n＜ウィンドウの数＞   ## 16x16タイル(別配布のタイルが必要)
  等で遊べます。

  また、スーパーユーザーとして setgid を付けてインストールしたい場合は、
  ./configure --with-setgid=games
  make
  の後、rootとして
  make install  
  を実行すれば、/usr/local/ 以下にインストールされます。
  インストール場所の変更等については ./configure --help を参照してください。

  標準の表示フォントはメインウィンドウが
  "-*-*-medium-r-normal--24-*-*-*-*-*-iso8859-1,-*-*-medium-r-normal--24-*-*-*-*-*-jisx0208.1983-0"
  で、サブウィンドウが
  "-*-*-medium-r-normal--16-*-*-*-*-*-iso8859-1,-*-*-medium-r-normal--16-*-*-*-*-*-jisx0208.1983-0"
  ですが、環境変数 ANGBAND_X11_FONT_? や ANGBAND_X11_FONT で変更できます。
  ANGBAND_X11_FONT_? は ?番目のウィンドウ(?は0から7)の設定になり、
  ANGBAND_X11_FONT は前者によって設定されなかったその他のウィンドウの設定になります。

  また、ウィンドウの表示位置やサイズを変える事もできます。
  ?番目のウィンドウ(?は0から7)について、
  ANGBAND_X11_AT_X_?でウィンドウのX位置、
  ANGBAND_X11_AT_Y_?でウィンドウのY位置、
  ANGBAND_X11_COLS_?でウィンドウのXサイズ(単位は半角の文字数)、
  ANGBAND_X11_ROWS_?でウィンドウのYサイズ(単位は半角の文字数)
  を設定できます。
